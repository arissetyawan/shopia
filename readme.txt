Functionality  in the project:
1- admin template in livewire and alpine js
2- QR code scanner
3- Barcode scanner
4- Large CSV file uploading Functionlity.
5- uploaded csv file in chunks using laravel queue jobs
6- show the file uploadeding progress bar in real time

Read the below steps to run the project.
1- composer update
2- npm install 
3- open the project in terminal 
4- php artisan migrate
5- php artisan db:seed
4- npm run dev
5- php artisan serve
6- http://127.0.0.1:8000
